Component: クラスコンポーネントを作成するための基底クラス。
PureComponent: shouldComponentUpdate を自動的に実装し、パフォーマンスを最適化するクラスコンポーネント。
memo: 関数コンポーネントをメモ化し、不要な再レンダリングを防ぐ。
useState: コンポーネントの状態を管理するための Hook。
useEffect: コンポーネントのライフサイクルイベント（マウント、更新、アンマウント）に対応する処理を実行するための Hook。
useContext: Context API を使用して、コンポーネントツリー全体でデータを共有するための Hook。
useRef: コンポーネントのライフサイクル全体で値を保持するための Hook。
useReducer: 複雑な状態管理ロジックを実装するための Hook。
useCallback: 関数をメモ化し、不要な再レンダリングを防ぐための Hook。
useMemo: 計算結果をメモ化し、パフォーマンスを最適化するための Hook。
イベント関連

SyntheticEvent: React が提供するイベントオブジェクトの基底クラス。
ユーティリティ

Children: 子コンポーネントを操作するためのユーティリティ関数群。
Fragment: 不要な DOM 要素を追加せずに複数の要素を返すためのコンポーネント。
createRef: DOM 要素や React コンポーネントへの参照を作成するための関数。
forwardRef: ref を子コンポーネントに転送するための関数。
lazy: コンポーネントを遅延読み込みするための関数。
Suspense: 遅延読み込みされたコンポーネントが読み込まれるまでの間、ローディング状態を表示するためのコンポーネント。
その他

StrictMode: 開発時に潜在的な問題を検出するためのコンポーネント。
Profiler: コンポーネントのパフォーマンスを測定するためのコンポーネント。
